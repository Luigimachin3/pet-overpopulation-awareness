This project is an educational webpage built with HTML, CSS, and JavaScript for the CodePath "Intro to Web Development" course. It raises awareness about pet overpopulation through interactive UI elements, dynamic content, and user engagement features.

## How to Run the Project
Some browsers restrict JavaScript features when opening files directly. For full functionality, run a local server:

python -m http.server 5500

Then open:

http://localhost:5500

## Features
- Light/Dark mode toggle  
- Dynamic header image via dog name selector  
- Book recommendations powered by the OpenLibrary API  
- Petition form with validation and auto-generated signatures  
- "Find Animal Shelters" shortcut (Google Maps)  
- "Share Pet Stories" button (Gmail compose)  
- Back-to-top button with smooth scrolling  
- Fixed navigation bar for quick section access  

## Technologies Used
HTML5  
CSS3  
JavaScript (ES6+)  
OpenLibrary Books API  
Python HTTP server (for local testing)

## About This Project
This webpage was created as part of CodePath's "Intro to Web Development" program and demonstrates concepts such as DOM manipulation, API integration, UI design, and interactive front-end behavior.
